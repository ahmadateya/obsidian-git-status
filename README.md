# Obsidian Git Status

A simple Obsidian plugin that visualizes git status directly in the file explorer.

## Features

- **File colors** — files in the explorer are colored based on their git status:
  - Red — untracked (new file not yet added to git)
  - Yellow — modified (tracked file with unsaved changes)
  - Green — staged (changes ready to commit)

## Requirements

- Desktop only
- Git must be installed and available in your system PATH
- Your vault must be inside a git repository

## Installation

1. Copy the plugin folder into your vault's `.obsidian/plugins/` directory
2. Enable the plugin in Obsidian settings under **Community Plugins**
